# RB-Y1 Robot Implementation

ROS 2 implementation of the Rainbow Robotics RB-Y1 robot with LIDAR, camera, SLAM, NAV2, and control systems.

## Features

- URDF model with original RB-Y1 meshes
- LIDAR integration
- Camera support
- SLAM implementation
- NAV2 navigation stack
- Robot controllers
